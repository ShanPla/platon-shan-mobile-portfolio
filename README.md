Mobile Portfolio App

A React Native + Expo mobile portfolio showcasing skills, projects, and contact information. Designed to be lightweight, responsive, and professional with dark/light mode support.

Features

Profile Section: Displays profile picture, name, and short bio.

Skills Section: Shows skills with color-coded progress bars indicating experience level.

Projects Section: Horizontal scrollable projects list with clickable modals containing details and GitHub links.

Contact Section: Email and social links.

Navbar: Sticky top navigation bar for quick scrolling to any section.

Theme Toggle: Light and dark mode toggle.

Footer: Simple footer with copyright or personal info.
